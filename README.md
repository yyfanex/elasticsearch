`yyfanex elasticsearch` 库包含了一系列与elasticsearch相关的通用工具，包括：
1、elasticsearch-template：与elasticsearch的增、删、改、查等方法模板化。
2、elasticsearch-sync：提供了elasticsearch与其他存储中间件的数据同步能力。

# 特性介绍

1. 操作模板化，简化了编程。
2. 同步能力支持。
